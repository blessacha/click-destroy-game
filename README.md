# Click Destroy Game

## Description

Click Destroy Game is a simple Unity game where the player clicks on cubes to make them disappear.

## Gameplay

* The scene contains several cubes on a platform.
* The player uses the mouse to click on the cubes.
* When a cube is clicked, it disappears from the scene.
* The goal is to destroy all the cubes.

## Controls

* Mouse Click → Destroy cube

## Features

* Click-to-destroy interaction
* Simple 3D Unity scene
* Multiple cube objects
* Basic object interaction using C#

## Built With

* Unity
* C#

## How to Run

1. Open the project in Unity.
2. Open the main scene.
3. Press Play.
4. Click the cubes to destroy them.
